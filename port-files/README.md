# FEZ

## Installation

1. **Download** `fez.zip` and extract it to your `ports/` folder.
2. **Copy game files** from your GOG FEZ installation into the `ports/fez/` folder (where `FNA.dll` is located):
   - `FEZ.exe`
   - `Content/` folder
3. **Launch** FEZ via PortMaster.

## Required Files from GOG

```
GOG FEZ Installation/
├── FEZ.exe              ← Copy this
└── Content/             ← Copy this entire folder
    ├── Art/
    ├── Fonts/
    ├── Music/
    ├── Sounds/
    └── ...
```

## Controls

| Button | Action |
|--------|--------|
| D-Pad | Move |
| A | Jump |
| B | Grab/Drop |
| X | Talk |
| L1/R1 | Rotate World |
| Start | Pause |
