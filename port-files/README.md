# FEZ

## Installation

1. **Download** this port via PortMaster
2. **Copy game files** from your GOG FEZ installation:
   - `FEZ.exe` → `ports/fez/`
   - `Content/` folder → `ports/fez/Content/`
3. **Launch** via PortMaster

## Required Files from GOG

```
GOG FEZ Installation/
├── FEZ.exe              ← Copy this
└── Content/             ← Copy this entire folder
    ├── Art/
    ├── Fonts/
    ├── Music/
    ├── Sounds/
    └── ...
```

## Controls

| Button | Action |
|--------|--------|
| D-Pad | Move |
| A | Jump |
| B | Grab/Drop |
| X | Talk |
| L1/R1 | Rotate World |
| Start | Pause |
